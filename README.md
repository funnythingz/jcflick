jQuery.jcflick
=======

iPhone, Android, iPad, Android3 タブレットなどのスマートフォン端末で、コンテンツをフリックできるようにするjQueryプラグインです。

レスポンシブ対応も考慮しているので、フリックイベントのないPCではカルーセルとして動作します。

メディアクエリ(MediaQueries)、横幅リキッドにも対応しています。
